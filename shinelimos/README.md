# Shine Limos Platform

Welcome to the Shine Limos project! This repository contains both the frontend and backend applications for the Shine Limos platform, a premium limousine booking and fleet management service.

## Project Structure

This repository is structured into two main applications:

- **/shinelimos** - The Frontend Application
- **/backend** - The Backend Application

---

### 1. Frontend (`/shinelimos`)

The frontend is a modern, responsive web application that provides the user interface for the platform, including the administrative dashboard for managing bookings and fleets.

**Key Technologies:**
- React with TypeScript
- Vite (for fast development and building)
- TailwindCSS (for styling)

**Getting Started:**
```bash
cd shinelimos
npm install
npm run dev
```

---

### 2. Backend (`/backend`)

The backend provides a robust RESTful API that powers the Shine Limos platform. It handles all business logic, data persistence, routing, and user authentication.

**Key Technologies:**
- Node.js
- Express.js

**Getting Started:**
```bash
cd backend
npm install
# Make sure your .env file is configured before starting the server
npm run dev
```
